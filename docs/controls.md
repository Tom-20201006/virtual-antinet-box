# Controls

## 鼠标

- 左键点击卡片或分隔片：选中并拿起。
- 左键拖动卡片或分隔片：在桌面平面上移动。
- 松开左键：放下；如果位于打开抽屉内，会吸附到抽屉内的简化排列位置。
- 左键拖动抽屉：沿固定方向开合。
- 右键拖动：旋转摄像机。
- 中键拖动：平移摄像机。
- 滚轮：缩放摄像机。

## 键盘

- `Q`：逆时针旋转当前卡片或分隔片。
- `E`：顺时针旋转当前卡片或分隔片。
- `R`：翻面。
- `O`：切换当前抽屉开合。
- `I`：导入本地图片，生成一张新卡片。
- `Ctrl+S` 或 `F5`：保存世界状态。
- `Ctrl+L` 或 `F9`：加载世界状态。
- `1` 或 `Home`：桌面总览。
- `2`：抽屉视角。
- `3` 或 `Space`：聚焦当前选中对象。

## 界面按钮

- “导入”：选择本地图片并生成新卡片。
- “保存”：保存当前世界状态。
- “载入”：加载上次保存的世界状态。
- “总览”：回到桌面总览视角。
- “抽屉”：切换到抽屉视角。
- “聚焦”：聚焦当前选中对象。

## 导入说明

导入流程只把图片作为卡片表面贴图使用。系统不会识别图片文字，不会 OCR，不会搜索，不会自动编号，不会根据内容推荐位置。
