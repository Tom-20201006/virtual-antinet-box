# Product Spec

## 产品定位

VirtualAntinetBox 是虚拟实体卡片盒，不是数字笔记软件。它只把实体桌面、卡片盒、抽屉、卡片和分隔片的物理承载层搬到 Godot 3D 空间中。

## 核心用户目标

- 在电脑中保留实体卡片盒的空间感、邻近关系和插入/取出体验。
- 让用户亲自判断卡片内容、编号、链接和放置位置。
- 用本地图片贴图还原扫描纸卡的视觉表面，但不让软件理解文字内容。

## 核心对象

- 桌面：临时摊开和整理卡片的工作表面。
- 卡片盒：承载抽屉的外壳。
- 抽屉：承载卡片顺序和邻近关系的物理容器。
- 卡片：扫描纸卡在虚拟空间中的实体薄片，有位置、厚度、朝向和碰撞。
- 分隔片：实体分隔片的虚拟对应物，不是软件标签。
- 导入图片：只作为卡片贴图资源。
- 本地状态文件：只保存物理状态和资源路径。

## 当前阶段目标

- 保持首版 Godot Demo 可运行。
- 保持最低限度数字化边界。
- 为后续根据真实 Antinet 工作流重建卡片盒交互做准备。
- 用 spec 和 research 分离决策与观察，避免把临时讨论写进当前状态。

## 当前阶段不做什么

- 不做 OCR。
- 不做搜索。
- 不做标签。
- 不做自动分类。
- 不做自动链接。
- 不做 AI 摘要。
- 不做知识图谱。
- 不做 Markdown 笔记编辑器。
- 不做云同步或账号系统。
- 不根据卡片内容推荐编号、位置或关系。
