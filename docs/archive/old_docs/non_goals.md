# Non Goals

以下不做项不是遗漏，而是项目原则。

## 明确不做

- OCR。
- 搜索。
- 标签。
- 自动分类。
- 自动链接。
- AI 摘要。
- 知识图谱。
- Markdown 笔记编辑器。
- 根据卡片内容生成编号。
- 根据卡片内容推荐位置。
- 数据库式知识管理。
- 云同步。
- 用户账号。
- 网页剪藏。
- 自动索引。
- 任何根据卡片文字内容进行智能处理的功能。

## 原因

本项目的目标是保存实体卡片盒中的位置、邻近、体积、插入、翻找和桌面摊开体验。内容理解越强，越容易把它变成数字笔记软件或知识管理系统。

VirtualAntinetBox 的边界是：只模拟物理环境，只保存物理状态和资源路径。
