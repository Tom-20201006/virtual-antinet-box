## 临时结论

- 新结构采用主控层、规格层、研究层、正式历史层、冷归档层。
- 主控层只保留当前状态、命令入口和当前任务缓冲。
- 当前状态文档最多保留最近 3 条关键变化，不保存完整历史。
- spec 记录“决定怎么做”，research 记录“观察到什么”。
- history 负责正式历史记录，archive 负责冷归档和旧文原文。
- 视频分析和物理组件分析不应与当前状态、项目目标、协作规则同层。
